## Overview
**LinksHQ** is a web link launcher designed for **Android-based car tablets** that don’t let you install PWAs or save websites as home screen shortcuts.

It’s built to be used in the car — especially on tablets — with a clean interface and large, clickable buttons. You can add any website you want, choose a custom icon, and launch it with one tap.

## Features
- Add your own websites
- Choose custom icons
- Car-friendly layout with big buttons
- Quick, simple launching
- Works without an internet connection
- Compatible with any car tablet running Android (as long as you can install apps)

> Example setups: BYD Atto 3, BYD SEAL, other Android-powered head units

## Why LinksHQ?
Because launching websites in your car **shouldn’t be this hard**.
